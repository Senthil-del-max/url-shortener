# Smart URL Shortener

A full-stack URL Shortener web application built using Spring Boot, MySQL, HTML, CSS, and JavaScript. This project allows users to shorten long URLs, create custom aliases, generate QR codes, and track URL usage through analytics.

## Live Demo

https://url-shortener-2itn.onrender.com

## Features

* Shorten long URLs instantly
* Create custom URL aliases
* QR Code generation for shortened URLs
* URL redirection support
* Click analytics tracking
* MySQL database integration
* Responsive user interface

## How It Works

1. User enters a long URL.
2. Optional custom alias can be provided.
3. Application generates a shortened URL.
4. QR Code is generated automatically.
5. Users can access the original URL using the short link.
6. Click counts are stored and tracked in the database.
